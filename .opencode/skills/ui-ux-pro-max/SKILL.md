## 功能描述

提供专业的 UI/UX 设计建议，包括：
- 布局设计建议
- 配色方案推荐
- 响应式设计指导
- 可访问性（a11y）检查
- 动效设计建议
- 设计系统构建

## 使用方法

```
帮我设计 [组件/页面] 的 UI
```

```
优化这个页面的用户体验
```

```
给我一个 [类型] 应用的设计方案
```

## 示例

```
帮我设计一个视频下载页面的 UI，要求简洁现代
```

```
检查这个表单的可访问性问题
```

```
推荐一个适合视频类应用的配色方案
```

## 设计原则

1. **简洁优先**：减少视觉噪音
2. **一致性**：统一的设计语言
3. **可访问性**：WCAG 2.1 AA 标准
4. **响应式**：适配多种设备
5. **性能**：减少动画卡顿
