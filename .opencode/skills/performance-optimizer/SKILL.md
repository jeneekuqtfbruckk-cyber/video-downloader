## 功能描述

性能分析和优化建议，包括：
- 代码性能分析
- 构建优化
- 运行时优化
- 内存泄漏检测
- 网络请求优化

## 使用方法

```
分析这个文件的性能问题
```

```
优化这个函数的执行效率
```

```
检查这个页面的加载性能
```

## 优化检查清单

### 前端性能

- [ ] 代码分割（Code Splitting）
- [ ] 懒加载（Lazy Loading）
- [ ] 图片优化（WebP、压缩）
- [ ] 缓存策略
- [ ] 减少 Bundle 大小
- [ ] Tree Shaking

### 后端性能

- [ ] 数据库查询优化
- [ ] 缓存策略（Redis）
- [ ] 异步处理
- [ ] 连接池配置
- [ ] 限流策略

### Node.js 性能

- [ ] 避免阻塞事件循环
- [ ] 使用 Stream 处理大文件
- [ ] 合理使用 Worker Threads
- [ ] 内存监控
