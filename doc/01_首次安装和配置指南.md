# 01_Warp + OpenCode + MiMo 首次安装和配置完整指南（Windows版）

> 更新时间：2026年5月8日

---

## 快速开始

本指南包含 **6 个步骤**，其中：
- ✅ **步骤 1-3**（安装 Node.js、Warp、OpenCode）→ 可用 `install.ps1` 脚本自动完成
- ⚠️ **步骤 4-6**（获取 API Key、配置、验证）→ 需手动完成

**一键安装**：
```powershell
# 以管理员身份运行 PowerShell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
.\install.ps1
```

---

## 步骤 1：环境检查 ✅ 自动

### 1.1 检查系统架构

```powershell
# 查看系统架构（应输出: AMD64 或 ARM64）
echo %PROCESSOR_ARCHITECTURE%

# 查看 Windows 版本（需要 Windows 10/11）
winver
```

### 1.2 检查前置依赖

```powershell
node --version    # 需要 v18+
npm --version     # 需要 9+
git --version     # 需要 2.0+
```

### 1.3 判断标准

| 工具 | 最低版本 | 推荐版本 | 未安装时 |
|------|---------|---------|---------|
| Windows | 10 | 11 | 无法使用 |
| Node.js | 18.0 | 20+ | 脚本自动安装 |
| npm | 8.0 | 10+ | 随 Node.js |
| Git | 2.0 | 最新 | 需手动安装 |

## 步骤 2：安装 Node.js ✅ 自动

```powershell
# 方法1: winget 安装（推荐）
winget install OpenJS.NodeJS.LTS

# 方法2: scoop 安装
scoop install nodejs

# 方法3: 手动下载
# 访问 https://nodejs.org 下载 LTS 版本
```

验证：
```powershell
node --version
npm --version
```

---

## 步骤 3：安装 Warp 终端 ✅ 自动

```powershell
# 方法1: winget 安装（推荐）
winget install Warp.Warp

# 方法2: 手动下载安装包
# x64: https://app.warp.dev/get_warp?package=exe_x86_64
# ARM64: https://app.warp.dev/get_warp?package=exe_arm64
```

验证：
```powershell
# 重启终端后
warp --version
```

> **注意**：Warp 安装后需要重启终端才能使用。

---

## 步骤 4：安装 OpenCode CLI ✅ 自动

```powershell
# 方法1: npm 全局安装（推荐）
npm i -g opencode-ai@latest

# 方法2: scoop 安装
scoop install opencode

# 方法3: chocolatey 安装
choco install opencode
```

验证：
```powershell
opencode --version
# 预期输出: opencode x.x.x
```

---

## 步骤 5：获取 MiMo API Key ⚠️ 手动

1. 访问小米 MiMo 平台：https://platform.xiaomimimo.com
2. 注册/登录账号
3. 进入「API Keys」页面
4. 点击「创建 API Key」
5. 复制生成的 Key（格式类似：`sk-xxxxxxxxxxxx`）

> **福利**：小米 MiMo 当前（2026年5月）正在免费发放 100 万亿 Token，截止 2026年5月28日。

## 步骤 6：配置 OpenCode 使用 MiMo ⚠️ 手动

### 6.1 配置 opencode.json

项目根目录已有 `opencode.json`，确认包含以下配置：

```json
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "xiaomi-mimo": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "Xiaomi MiMo",
      "options": {
        "baseURL": "https://platform.xiaomimimo.com/v1"
      },
      "models": {
        "MiMo-V2.5-Pro": {
          "name": "MiMo V2.5 Pro"
        },
        "MiMo-V2.5": {
          "name": "MiMo V2.5"
        }
      }
    }
  }
}
```

### 6.2 使用 /connect 命令配置

在 Warp 中启动 OpenCode：

```powershell
# 进入项目目录
cd D:\Xcode\20260503_去水印

# 启动 OpenCode
opencode
```

在 OpenCode TUI 中执行：

```
/connect
```

选择「Xiaomi MiMo」或手动输入 API Key。

### 6.3 配置环境变量（可选）

项目根目录已有 `.env.local`，确认包含：

```env
# MiMo API Key
MIMO_API_KEY=sk-xxxxxxxxxxxx
```

## 常见错误及解决方法

### 安装错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| `npm: command not found` | Node.js 未安装 | 运行 `install.ps1` 或手动安装 |
| `opencode: command not found` | OpenCode 未全局安装 | `npm i -g opencode-ai@latest` |
| `permission denied` | 权限不足 | 以管理员运行 PowerShell |
| `EACCES` 错误 | npm 权限问题 | 配置 npm 全局目录或使用 nvm-windows |

### 连接错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| `API Key invalid` | Key 错误或过期 | 重新获取 Key |
| `Connection refused` | 网络问题 | 检查代理/VPN |
| `Rate limit exceeded` | 频率限制 | 等待或升级套餐 |
| `Model not found` | 模型名称错误 | 运行 `/models` 查看可用模型 |

### Warp 相关错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| Warp 启动失败 | GPU 驱动问题 | 更新显卡驱动 |
| 终端卡顿 | 渲染问题 | 关闭 GPU 加速 |
| 快捷键冲突 | 键位冲突 | 修改 Warp 快捷键设置 |

### 项目配置错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| `opencode.json 无效` | JSON 格式错误 | 使用 JSON 校验器 |
| `Provider not found` | 提供商配置错误 | 检查 `opencode.json` |
| `Module not found` | 依赖缺失 | `npm install` |
